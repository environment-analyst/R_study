# 1. select
* 특정 컬럼만 가져오기-> 정보 선택

## 01.01. 특정 컬럼만 가져오기
```R
df %>%
  select(컬럼명1,컬럼명2)
```

## 01.02. 특정 컬럼만 제외하고 가져오기
```R
df %>%
  select(-컬럼명1,컬럼명2)
```

## 01.03. 컬럼명이 넘 길면 위치(숫자)로 가져올 수 있음
```R
df %>%
  select(3,4)
```

## 01.04. 컬럼명의 시작 글자로 가져올 수 있음
```R
df %>%
  select(strars_with('0))
```

## 01.05. 컬럼명의 끝 글자로 가져오기
```R
df %>%
  select(ends_with('nm'))
```

## 01.06. 컬럼명의 포함글자로 컬럼불러오기
```R
df %>%
  select(contains('t'))
```
